定例会に参加できた方は以下 1 点の対応をお願いします。

``` console
（１） 感想フォームへの回答
（２） 目標振り返りワーク結果の送付
```

> ※注1: すでに感想フォームへの回答済みの方は対応不要です
> 
> ※注2: 途中から定例会に参加された方は、定例会の録画を視聴してからの感想フォーム回答をお願いします

## 途中参加の方は

定例会に途中参加された方は、参加できていなかった部分について、以下の録画内容を確認の上、感想フォームへの回答をお願いします。

<p>
<strong>2021年度7月度 本社定例会</strong>（録画）:<br />
<a href="https://youtu.be/9S80uMfuHmI" target="_blank">https://youtu.be/9S80uMfuHmI</a>
</p>

## （１） 感想フォームへの回答

以下のアンケートフォームからアンケートへの回答記載、送信をお願いします。

<p>
<strong>7月度定例会 感想フォーム</strong>（当日参加者向け）:<br />
<a href="https://forms.gle/V1HTpB6ghHPnUtFm9" target="_blank">https://forms.gle/V1HTpB6ghHPnUtFm9</a>
</p>

アンケートの回答ができた時点で、定例会への出席とみなします。

### アンケート締め切り

7/14(水) 23:59 まで

## （２） 目標振り返りワーク結果の送付

また、当日参加できた方は、    
目標振り返りワークが終わった後の Excel ファイルは、メールで人事部あてに提出してください。

### Excel ファイルの提出先

> **メールアドレス：** ![提出先メールアドレス](/fs-monthly-meeting-2021-07/svg/address-for-submission.svg)
> 
> **件名：** 「目標振り返り結果：7月度定例会」
> 
> **Excel ファイル名：** 「目標振り返り.2021.07_氏名.xlsx」

#### Excel ファイルの提出期限

7/14(水) 23:59 まで

以上、よろしくお願いいたします。
